# Project name：颜究院Yanlab AI (wechat app)

一个基于微信小程序的 AI 皮肤分析与轻医美建议平台。

## ✨ 功能概览

- AI 拍照皮肤检测
- 个性化护肤建议
- 美学评分与建议
- 探索穿搭、护肤品推荐

## 🛠 技术栈

- 前端：WeChat MiniProgram（Skyline + glass-easel）
- 后端：FastAPI（Python）
- AI 分析：OpenCV + TensorFlow
- 数据库：MySQL
